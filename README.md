# FIT2179 Data Visualisation 2

This project visualises Australia's tourism recovery using Australian Bureau of Statistics visitor arrivals data and BITRE airport traffic data.

The final webpage is built with HTML, CSS and Vega-Lite.

## Vega-Lite specifications

Readable Vega-Lite JSON specifications are included in the `specs/` folder. The webpage embeds the same visualisations through `js/vega_lite_vis.js`.

## Data sources

- Australian Bureau of Statistics, Overseas Arrivals and Departures, Australia
- Bureau of Infrastructure and Transport Research Economics, Airport Traffic Data
